# 音乐推荐系统 python513

## 项目概述

音乐推荐系统是一款结合了Python后台开发与前端可视化展示的应用。该系统以Flask为后端框架，VUE实现前端展示，提供用户友好的音乐推荐服务。

## 技术栈

- 后端：Python、Flask
- 前端：VUE、ECharts

## 功能模块

- 用户登录
- 歌单推荐
- 音乐数据可视化分析

## 系统角色

- 普通用户：可登录系统，查看推荐歌单，进行音乐试听。

## 运行环境

- 后端：Python 3.x，Flask
- 前端：VUE，现代浏览器
- 数据库：视具体配置而定

## 部署步骤

具体部署步骤请参考项目内的部署教程。

## 目录结构

```
音乐推荐系统/
├── backend/                # 后端代码目录
│   ├── app/                # Flask应用目录
│   ├── models/             # 数据模型目录
│   └── ...
└── frontend/               # 前端代码目录
    ├── assets/             # 资源文件夹
    ├── components/         # 组件目录
    └── ...
```

## 核心亮点

- **深度学习与协同过滤**：结合深度学习技术和协同过滤算法，为用户提供精准歌单推荐。
- **数据可视化**：使用ECharts进行数据可视化，直观展示音乐数据。
- **数据集丰富**：含3万条音乐数据，提供丰富推荐依据。

## 数据集说明

项目包含3万条音乐数据集，用于训练推荐模型和提供用户推荐服务。
